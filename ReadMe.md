# Simple Discord XP Bot

## Replit Setup
Install Modules
``` terminal
npm install
npm install ts-node --global
```

Create A New File Called .env
``` .env
    TOKEN="YourBotToken"
    REPLIT="yes"
```
To Start Run
``` terminal
npm run start
```
## Python Setup
``` terminal
npm install ts-node --global
python3 Start.py
```
